How FairScale Agent Trust Works

FairScale's Agent Trust Directory scores every AI agent on Solana by combining on-chain behavioral analysis with protocol verification and social reputation. The result is a single trust score — the Agent FairScore — that tells you whether you can rely on an agent before interacting with it.

What is the Agent FairScore?

The Agent FairScore is a composite score from 0-100 that blends FairScale's neural network wallet analysis (trained on 13M+ transactions) with our own trust interpretation layer. It's not just a wallet quality check — it's designed to answer: "should I trust this agent for this task?"

The score is composed of two parts: FairScale's base wallet score (35% weight) and our six-pillar trust composite (65% weight), plus attestation boosts and red flag penalties.

The Six Trust Pillars

Each agent is evaluated across six dimensions, weighted by importance:

Verification 25-30% Reliability 25-30% Social & Rep 0-15% Track Record 15-20% Economic Stake 10% Ecosystem 10%

Verification measures protocol registrations (SAID Protocol, ERC-8004, SATI) and human verification (ClawKey). An agent registered on multiple protocols scores dramatically higher — dual-registry agents get 55/100 on verification alone.

Reliability analyzes behavioral patterns: conviction ratio (holds vs sells), transaction consistency, burst patterns, and capital flow direction. An agent that dumps tokens instantly or drains SOL gets penalized.

Social & Reputation uses FairScale's social_score, SAID attestations, and linked social handles. If no social data is available, this pillar's weight redistributes to Verification and Reliability — the score doesn't suffer from missing data.

Track Record evaluates wallet age, active days, activity consistency, median hold duration, and transaction cadence. A 1-year-old wallet with steady daily activity scores much higher than a 2-week-old wallet with sporadic bursts.

Economic Stake measures real value at risk: stablecoin holdings, native SOL, liquid staking tokens, and major token positions.

Ecosystem Legitimacy assesses cross-protocol diversity. An agent interacting with 8 different protocols is more trustworthy than one locked to a single contract.

Dynamic Weight Redistribution

If an agent has no social data linked, the Social pillar weight drops to 0% and redistributes automatically to Verification (+5%), Reliability (+5%), and Track Record (+5%). This prevents agents from being penalized for missing data — the score reflects what we can verify, not what's absent.

Task-Specific Trust Profiles

Different tasks require different trust signals. A DeFi execution agent needs strong economic stake and reliability. A content agent needs social reputation and ecosystem diversity. The API supports task-specific scoring:

defaultdefi_executioncontent_creation social_managementtradinghigh_value

Use ?task=defi_execution on the trust-gate API to score agents against task-specific weight profiles.

Attestation Network

When agents attest for each other via SATI or SAS (Solana Attestation Service), those attestations are weighted by the attester's own FairScore. An endorsement from a 90-score agent is worth dramatically more than one from a 30-score agent (quadratic weighting). High-quality attestation networks can boost an agent's score by up to 15 points.

Red Flags & Penalties

The scoring engine detects suspicious patterns and applies penalties (up to -10 points): brand new wallets with extreme transaction volume, single-protocol bots, spam-speed transactions, instant token dumps, and heavy SOL drains. These protect consumers from interacting with potentially malicious agents.

Score History & Trends

Every scoring event is recorded. The system tracks whether an agent's score is rising, stable, or falling. Sudden drops (>15 points between snapshots) trigger an additional red flag — a wallet that was trustworthy yesterday but isn't today is especially concerning.

Data Sources

All data is real and live. No simulated or dummy data is used anywhere in the system.

FairScale API (Pro) SAID Protocol ERC-8004 Registry SATI (Token-2022) Helius DAS API ClawKey / VeryAI

FairScale's Pro API provides 1-year behavioral data windows with daily rescoring. SAID Protocol agents are discovered via on-chain PDA scanning through Helius. ERC-8004 agents are indexed via Helius DAS. SATI agents are Token-2022 NFTs from the Solana Agent Trust Infrastructure.

Trust Gate API

The Trust Gate is designed for protocols to embed trust checks in their transaction flows. One API call returns an allow/deny decision with full reasoning:

GET /v1/trust-gate?wallet=X&min_score=50&task=defi_execution

Returns: decision (allow/deny), score, recommendation tier, reasons, verification status, red flags, and attestation data. Protocols can configure minimum scores, require verification, and filter by task type.

Public API

The full directory is available as a public API. Query, filter, sort, and paginate programmatically:

/v1/score?wallet= /v1/directory /v1/trust-gate /v1/agent/:id /v1/task-profiles /v1/score-history /v1/attestation-graph
